# Contributing to ISDCProtocol2025 Documentation

How to contribute
- If you have access: edit the wiki directly (push to the wiki git repository https://github.com/isystemsdirect/ISDCProtocol2025.wiki.git).
- If you do not have wiki access: open issues or PRs in the main repo proposing changes to the docs or the README.

Style guidelines
- Use clear examples, include both header JSON and body
- Keep wire-level examples copy-pasteable
- When adding breaking changes, update Migration Guide and Changelog
